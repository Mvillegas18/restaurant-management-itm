# restaurant-management-itm

## 📌 Descripción del Proyecto

Este proyecto consiste en el desarrollo de un **sistema de gestión de restaurante**, creado como parte de un trabajo académico del Instituto Tecnológico Metropolitano (ITM) de Medellín.

La aplicación permite administrar procesos clave del negocio como la gestión de pedidos, clientes y productos, facilitando el control y organización de la información de manera eficiente.

El sistema está desarrollado bajo una **arquitectura en 3 capas**, separando responsabilidades en:
- **Domain**: lógica de negocio y entidades principales
- **DataAccess**: acceso y persistencia de datos
- **Web API**: exposición de servicios mediante una API REST

El backend está construido con **.NET**, siguiendo buenas prácticas de desarrollo y principios de arquitectura limpia, mientras que el frontend se desarrolla con **Angular**, proporcionando una interfaz moderna, dinámica y fácil de usar.

Este proyecto tiene como objetivo aplicar conocimientos de desarrollo **fullstack**, diseño de APIs REST y estructuración de aplicaciones escalables y mantenibles.
